# SmartMath

Sebelum siswa mengakses E-LKPD, sistem SmartMath melakukan asesmen diagnostik dan identifikasi preferensi belajar. Data tersebut dianalisis menggunakan algoritma K-Means Clustering untuk menghasilkan kelompok belajar yang sesuai. Selanjutnya, siswa diarahkan pada E-LKPD yang telah dipersonalisasi berdasarkan kebutuhan masing-masing.
